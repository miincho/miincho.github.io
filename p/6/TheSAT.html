The SAT
A web-based personality-based custom avatar generator. Includes a 3D experience dedicated to finding out personal details about you to create your own custom avatar.
Timeline: April-May 2023
Programs: p5.js, HTML/CSS
Role: Web Designer, Front-end
Concept
NYC OpenData
The current state of avatar creation can be customized in a way that we want to be perceived. I wanted to create a more accurate way to generate an avatar that focuses more on personality/psychology. In this site, I focused on 3 different personal traits: fear, love, and regret.
