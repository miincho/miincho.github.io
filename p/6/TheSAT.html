THE SAT
This website combines data from SAT test scores alongside the racial demographics of each school, this site acts to visualize the socio-economic advantages in standardized testing.
Timeline: April-May 2023
Programs: p5.js, HTML/CSS
Role: Web Designer, Front-end
CONCEPT
NYC OPENDATA
The green version of the site displays the SAT scores in which each score can be clicked to reveal a short test. You can challenge the SAT score that you clicked on and try to score higher.

The pink, or inverted, version displays the same SAT scores, but also shows the racial demographics of the school you click on. You can take a modified version of the test where the difficulty scales with what race makes up the majority of the school.
Data of average test scores and the racial demographics
PROCESS
DESIGNING THE SITE
The landing page was created to look like the perspective of a student about to take a test. It shows the admission ticket with various test taking items scattered across the screen.
Various versions of 2D interfaces
DATA VISUALIZATION
Each of the flickering numbers will stop on the average test score once the cursor is hovered over it. The size of the number is determined by the population size of each school. Clicking a number will display the name of the school and the racial demographics.

The numbers are displayed in columns to represent the answer forms used for standardized testing.
